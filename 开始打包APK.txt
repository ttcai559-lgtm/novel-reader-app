┌────────────────────────────────────────────────────────────┐
│  第1步：创建GitHub账号（2分钟）                             │
└────────────────────────────────────────────────────────────┘

访问：https://github.com/signup

输入：
  - 邮箱
  - 密码
  - 用户名

验证邮箱 → 完成！

已有账号？直接登录：https://github.com/login


┌────────────────────────────────────────────────────────────┐
│  第2步：创建仓库（1分钟）                                   │
└────────────────────────────────────────────────────────────┘

1. 登录后点击右上角 "+" → "New repository"

2. 填写：
   Repository name: novel-reader-app
   Privacy: ✅ Private（私有）

3. 点击 "Create repository"

4. 记下仓库地址


┌────────────────────────────────────────────────────────────┐
│  第3步：上传代码（3-5分钟）                                 │
└────────────────────────────────────────────────────────────┘

方法A：GitHub Desktop（推荐，图形界面）
  1. 下载：https://desktop.github.com/
  2. 安装并登录
  3. File → Add Local Repository
  4. 选择：D:\Python_file\APP_Tool
  5. Publish repository

方法B：命令行
  打开PowerShell，运行：

  cd D:\Python_file\APP_Tool
  git init
  git add .
  git commit -m "Initial commit"
  git remote add origin https://github.com/你的用户名/novel-reader-app.git
  git branch -M main
  git push -u origin main


┌────────────────────────────────────────────────────────────┐
│  第4步：触发构建（1分钟）                                   │
└────────────────────────────────────────────────────────────┘

1. GitHub仓库页面 → "Actions" 标签

2. 左侧 → "Build Android APK"

3. 右侧 → "Run workflow" → "Run workflow"

4. 构建开始！⏳


┌────────────────────────────────────────────────────────────┐
│  第5步：等待完成（25-30分钟，自动进行）                     │
└────────────────────────────────────────────────────────────┘

可以去做其他事，等完成后：

✅ 绿色勾 = 成功
❌ 红色叉 = 失败（告诉我错误信息）


┌────────────────────────────────────────────────────────────┐
│  第6步：下载APK（2分钟）                                    │
└────────────────────────────────────────────────────────────┘

1. Actions页面 → 点击构建任务

2. 向下滚动 → "Artifacts" 部分

3. 点击 "novelreader-apk" 下载

4. 解压ZIP → 得到APK文件


┌────────────────────────────────────────────────────────────┐
│  第7步：安装到手机（2分钟）                                 │
└────────────────────────────────────────────────────────────┘

1. APK传到手机（微信/QQ/数据线）

2. 手机上点击APK

3. 允许安装 → 完成！
